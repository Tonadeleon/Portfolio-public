When I learned about regression models, my teacher taught us by using simulation modeling. He made up a game for us to create a simulated data set with specific parameters to create any laws/models that we wanted. Then put us to compete against each other and see who could figure out the other studentâ€™s model by using regression techniques.

Here Iâ€™ll show you one of my models with which I competed in the game called Regression-Battleship. This model started as a simple representation of a spiral galaxy, but quickly ended up turning into a different (but still good) idea!
